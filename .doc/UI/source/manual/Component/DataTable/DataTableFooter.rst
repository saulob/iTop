The columns (*aColumns*) have the following format:

::

    [
        'nameField1' => ['label' => labelField1, 'description' => descriptionField1],
        ...
    ]

The data (*aData*) format has to be:

::

    [
        ['nameField1' => valueField1, 'nameField2' => valueField2, ...],
        ...
    ]
